- [技術評論社 (電子書籍版もあり)](http://gihyo.jp/book/2016/978-4-7741-8066-3)
- [Amazon](www.amazon.co.jp/dp/4774180661)

ネタ元: [SQL書き方ドリルのサンプルDBをre:dashで遊べるようにするVagrantfile](http://ariarijp.hatenablog.com/entry/2016/09/05/140559)
